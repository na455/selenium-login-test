# Selenium Login Basic Test

## Description
This is a simple Selenium Java automation project for testing the login page on **The Internet (Herokuapp)**.  
The project is created for practicing basic **WebElement actions** such as:

- `sendKeys()` – to enter text  
- `clear()` – to clear text fields  
- `click()` – to press buttons  
- `getText()` – to read messages  

## Website Used for Testing
[The Internet - Login Page](https://the-internet.herokuapp.com/login)

## Steps Performed in the Test
1. Open the login page  
2. Enter username (`tomsmith`)  
3. Enter password (`SuperSecretPassword!`)  
4. Click **Login** button  
5. Click **Logout** button  
6. Print the message shown after login/logout  

## How to Run
1. Clone the repository  
2. Open in IntelliJ IDEA  
3. Make sure Maven dependencies are loaded (`selenium-java` and `webdrivermanager`)  
4. Run `basictest.java`
